# Smart-Analyzer--Python-ProjectSmart Analyzer 🚀
Overview

Smart Analyzer is a Python-based GUI automation tool developed to simplify Excel and CSV data cleaning, transformation, aggregation, and file management tasks through an interactive desktop application.

The application allows users to clean datasets, combine multiple files, perform aggregations, preview processed data, and export results efficiently without writing manual scripts.

Features
📂 File Handling
Browse and load Excel/CSV files
Read and process datasets dynamically
Combine multiple Excel/CSV files into a single workbook with different sheets

🧹 Data Cleaning
Remove duplicate records
Remove blank spaces
Remove leading and trailing spaces
Convert columns into text format

📊 Data Aggregation
Built-in aggregation functions for:
Numeric columns
Text columns
Generate summarized outputs easily

👀 Data Preview
Preview data after applying filters or aggregation
Real-time visibility of processed data

🔄 Reset Functionality
Reset filters
Reset aggregation settings
Restore workflow quickly

📤 Export Functionality
Export cleaned and processed data
Save output files to desired location
